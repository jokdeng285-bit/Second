# Spark — Dating App Website

> *Real connections built on shared interests, not just first impressions.*

A fully static, multi-page dating app website built with HTML, CSS, Bootstrap 5, and vanilla JavaScript. No frameworks, no build tools — open `index.html` in a browser and it works.

---

## Pages

| File                   | Route              | Description                                                                                  |
| ---------------------- | ------------------ | -------------------------------------------------------------------------------------------- |
| `index.html`           | `/`                | Homepage — hero with animated heart particles, features, live match strip, testimonials, CTA |
| `how-it-works.html`    | `/how-it-works`    | Step-by-step sign-up flow + FAQ accordion                                                    |
| `browse.html`          | `/browse`          | Profile grid with interest filters + name search                                             |
| `pricing.html`         | `/pricing`         | Three-tier pricing with monthly/annual billing toggle                                        |
| `success-stories.html` | `/success-stories` | Couple story cards with photos and quotes                                                    |
| `blog.html`            | `/blog`            | Featured post + 6-card article grid                                                          |
| `about.html`           | `/about`           | Mission, values, company timeline, team cards                                                |
| `contact.html`         | `/contact`         | Validated contact form with success state                                                    |

---

